## Notes

- `00` is intentionally zero-indexed so **NemoClaw** stays the silent prime mover.
- `03 Paperclip` is in but flagged — unresolved question whether paperclips count as claws. See it as a cautionary member.
- Any of the `proposed` entries can be promoted by scaffolding a `claude/setup-<name>-XXXX` branch, mirroring the existing pattern used for NemoClaw / OpenClaw / InstantlyClaw / FireClaw.
